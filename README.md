# Next Action Audit

A compact Next.js 14 App Router application used as source data for authentication-coverage tooling.

- `app/` contains route groups, pages, layouts, and route-local actions.
- `src/actions/` contains domain Server Actions.
- `src/components/` contains reusable UI components.
- `src/lib/` contains Prisma access, repositories, and shared helpers.
- `prisma/` contains the data model.

The application uses TypeScript, NextAuth session lookup, and Prisma repositories.
